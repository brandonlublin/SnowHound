## How to Use

1. **Select Location**: Use the search bar to find a location, or use your current location
2. **Choose Models**: Select 1-5 weather models to compare (default: GFS and ECMWF)
3. **View Data**: Switch between Graph, Table, and Chart views
4. **Explore Details**: Click on any day in the 7-day forecast to see hourly breakdowns
5. **Save Favorites**: Click the star icon to save locations for quick access
6. **Share Forecasts**: Use the share button to share forecasts with others

## Weather Data Sources

SnowHound aggregates data from multiple weather APIs:
- **OpenWeatherMap** - Global weather data
- **WeatherAPI.com** - Comprehensive forecasts
- **National Weather Service (NWS)** - US weather data

## Tech Stack

Built with React 19, TypeScript, Vite, Tailwind CSS, and Recharts. Features a secure backend with MongoDB caching and rate limiting.
